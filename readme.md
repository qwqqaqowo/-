项目介绍
  本项目是基于 Spring Boot + MyBatis + Thymeleaf 开发的在线留言板系统，实现了用户注册、登录、发帖、评论、帖子分页搜索、权限管理（普通用户 / 管理员）等核心功能，界面采用统一的现代化风格设计，适合作为 Java Web 实习项目或课程设计。
技术栈
  后端：Spring Boot 2.7.15、MyBatis、MySQL 8.0
  前端：Thymeleaf、HTML5、CSS3、原生 JavaScript
  构建工具：Maven

环境准备
  JDK：1.8 及以上
  MySQL：8.0 及以上
  Maven：3.6 及以上
  IDE：IntelliJ IDEA/Eclipse（推荐 IDEA）
快速启动
  克隆 / 下载项目将项目代码导入 IDE，确保 Maven 自动下载依赖。
